# Examples

Real-world usage examples for Guitar Tools.

This directory contains example VCV Rack patches and configuration guides demonstrating various use cases.

Simple guitar processing chain suitable for beginners.

**Contents:**

- Basic tone shaping

**Difficulty:** Beginner

#### [clean-tone-setup.md](clean-tone-setup.md)
Creating crystal-clear clean guitar tones.

**Contents:**

- EQ settings for sparkle

- Reverb and modulation

**Difficulty:** Beginner

#### [high-gain-metal.md](high-gain-metal.md)
Modern high-gain metal tone with tight low end.

**Contents:**
- 5150/6505 model setup
- Noise gate configuration
- Aggressive EQ curve
- Tight palm muting

**Difficulty:** Intermediate
